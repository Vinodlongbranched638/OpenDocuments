## 🖥️ What OpenDocuments Does

OpenDocuments helps you search across your files and apps with plain questions.

Use it to connect tools like GitHub, Notion, and Google Drive. Then ask a question and get answers with source links so you can check the result.

It is made for people who want a single place to find information without digging through many tabs.

## 🧭 First-Time Setup

After you open OpenDocuments for the first time, you can set up your data sources.

### Add a source

Choose one of these:

- GitHub
- Notion
- Google Drive

Then sign in to the account you want to use and let OpenDocuments read the content you choose.

### Choose your AI provider

OpenDocuments can work with:

- Ollama for local use
- OpenAI for cloud use
- Claude for cloud use

If you want to keep things local, use Ollama on your computer. If you want a hosted option, connect OpenAI or Claude.

### Test your setup

After setup, ask a simple question like:

- What files mention the budget?
- Where is the project plan?
- What did we decide about the launch date?

If the answer shows source links, the setup is working.
